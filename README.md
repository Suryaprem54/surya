# 🏦 ATM Simulator (Java)

This is a simple **ATM Simulator** program written in **Java**. It allows users to:
- ✅ Check their account balance
- ✅ Deposit money
- ✅ Withdraw money
- ✅ Securely verify their PIN before transactions

---

## 🚀 Features
- **Object-Oriented Design**: Uses `BankAccount` and `ATM` classes.
- **Secure PIN Verification**: Ensures only authorized users access the account.
- **Transaction Handling**: Deposits and withdrawals with proper balance validation.
- **User-Friendly Interface**: Uses **Scanner** for input in the console.

---

## 🛠️ Technologies Used
- **Java** (Core Java, OOPs concepts)
- **Scanner Class** (for user input)
- **GitHub** (Version control)
